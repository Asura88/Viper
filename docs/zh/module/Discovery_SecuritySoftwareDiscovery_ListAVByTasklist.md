# 查找杀毒软件进程

# 主要功能
模块通过将 tasklist /svc 与已知数据对比,获取系统已安装杀毒软件信息.

通过webshell命令执行功能执行 tasklist /svc ,然后将结果拷贝到输入框运行即可

**<font style="color:#00b580;"></font>**

# 操作方法
+ 使用webshell执行

```plain
tasklist /svc
```

+ 运行模块,将结果填入
