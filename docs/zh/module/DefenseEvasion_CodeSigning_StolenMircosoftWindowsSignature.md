# 伪造Microsoft Windows签名

# 主要功能
在未签名的EXE文件中添加Microsoft Windows签名 

# 注意事项
签名效果如下,无法通过操作系统签名认证

![1623379420996-c59b14e2-3bad-4181-b34f-9eb990b6eca9.webp](./img/9zk7VQ5JF6Q7VMaf/1623379420996-c59b14e2-3bad-4181-b34f-9eb990b6eca9-987183.webp)

# 操作方法
+ 上传需要添加签名的exe到`文件管理`
+ 运行模块
+ `文件管理`中会生成已添加签名的文件
