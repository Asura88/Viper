# 伪造Microsoft Windows签名

在未签名的EXE文件中添加Microsoft Windows签名

## 注意事项

签名效果如下,无法通过操作系统签名认证

![](img\DefenseEvasion_CodeSigning_StolenMircosoftWindowsSignature\1.webp)

## 操作方法

+ 上传需要添加签名的exe到`文件管理`
+ 运行模块
+ `文件管理`中会生成已添加签名的文件

![](img\DefenseEvasion_CodeSigning_StolenMircosoftWindowsSignature\2.webp)
