# Reverse_TCP_RC4直连免杀

生成免杀exe及其源码

## 操作方法

+ 新建reverse_tcp_rc4监听
+ 打开模块,选择对应的rc4监听
+ 运行模块后即会在`文件管理`中生成一个zip文件,包含免杀的exe载荷及对应的C++代码
+ 可使用visual studio自定义修改及编译源码
