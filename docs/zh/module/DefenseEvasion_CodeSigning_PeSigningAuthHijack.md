# 劫持Windows数字签名认证

# 主要功能
修改Windows默认数字证书认证文件,使系统默认认证所有数字签名有效. 模块会影响系统所有exe的运行签名认证,请谨慎使用.



# 操作方法
+ 选择session
+ 执行模块
