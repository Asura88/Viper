# Windows 增加本地账户/域账户

# 主要功能
通过Windows API添加用户到本地计算机或域

# 注意事项
填写用户组时需要注意Session必须有该用户组的操作权限,添加域用户时要注意Session所属用户必须在域中,并有对应用户组的操作权限.
