# Windows WDigest开关

# 主要功能
用于打开WDigest凭证存储开关,Windows 8/2012以上版本默认关闭Wdigest开关.

通过修改注册表UseLogonCredential值打开开关.

当用户注销重新登录后即可使用mimikatz抓取密码.<font style="color:rgba(255, 255, 255, 0.85);">.</font>
