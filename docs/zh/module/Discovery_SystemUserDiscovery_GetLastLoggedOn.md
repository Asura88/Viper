# 获取域主机最后登录用户

# 主要功能
模块获取域内远程主机最后登录的用户信息. 模块需要普通的域用户权限,但是需要远程主机开启远程注册功能. 主机名可以使用<<获取所有域主机的信息>>模块进行获取
