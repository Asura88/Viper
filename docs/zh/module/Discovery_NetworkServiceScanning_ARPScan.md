# 内网ARP扫描

# 主要功能
目标内网的ARP扫描,所有ARP请求与回复目标内网. 

ARP消息只证明此主机存活,不会探测端口. 如其他模块需要连接发现的主机,请使用Session添加对应路由或Socks代理.

请注意大部分ARP消息局限于同一个路由器下面,也就是说ARP扫描不能跨路由器.
