# 执行自定义Python脚本

# 主要功能
主机内存中(Windows)或者系统Python解释器(Linux)执行自定义的Python脚本. 

Python脚本只支持Python2.7及其自带的库. 

执行的脚本可以通过`文件列表`上传到服务器
