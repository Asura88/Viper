# 内网端口扫描与服务识别

# 主要功能
扫描内网中开放的TCP端口,识别已发现端口的服务,软件,版本等 

所有扫描的网络流量在目标内网, 如其他模块需要连接发现的服务,请添加对应路由及Socks代理
