# 异步Netbios扫描

# 主要功能
基于nextnet的内网异步netbios扫描. 

模块扫描子网开放netbios协议(137端口)的主机,并获取主机的网卡地址. 

模块主要用于在拥有多个网段的内网中寻找目标.
