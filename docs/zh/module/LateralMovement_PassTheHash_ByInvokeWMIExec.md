# Invoke-WMIExec哈希传递

# 主要功能
使用已知的用户名及NTLM哈希,通过wmi方式在目标主机执行载荷.

模块使用内存执行Invoke-WMIExec方式执行wmi,相较于<WMI哈希传递>无需上传文件
